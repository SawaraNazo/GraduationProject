# 项目介绍
引擎及版本：Cocos2d-x 3.16  
语言：C++  
软件：Microsoft Visual Studio Community 2017  

# 游戏简介
大富翁是一种历史悠久、时代感强的游戏类型，本游戏的本质就是地产的争夺。通过掷骰前进，触发不同的事件，获得/支出身上的资金，最后能够笑到最后的玩家将是这个游戏的最终赢家。  
游戏主要分为以下模块：标题动画、主界面、主游戏流程、游戏设置、进度存储和读取。
